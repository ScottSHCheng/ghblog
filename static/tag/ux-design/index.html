UX Strategy 剝去產品設計者的面子問題

感謝我的前同事 Eason 推薦這本好書給我！ 作者 Jamie Levy 真的是把創業者對於產品設計的心路歷程寫到心坎裡。邊看也邊讓我自己細數創業過程所走過的冤枉路，學到的教訓，和心態上的誤區(我稱之為創業者的大頭症)。 創業者的大頭症 在第三章驗證價值主張提到：創作產品當然要認識你的顧客。恩...這聽起來不是理所當然的(廢話)嗎？ 但事實是，很少產品設計者這麼做 =.=。 摘錄書中一段銘心刻骨的觀察： 認識顧客是一個用來發現、測試和驗證產品是否能幫一群使用者族群解決某個大問題的過程，基本上就是進行使用者研究。但你不是只觀察和同理，然後下結論而已，不要閉門造車，走出去做使用者驗證，落實精實創業的基本方法。你必須主動聆聽人們的聲音並打好關係，因為你的目的是要找到他們欲解決的問題。 這聽起來好像很理所當然，但令人驚訝的是，我在新創和企業中所合作過大多數的利害關係人是沒有在和顧客…
